# Hello world program!  

> download repository  
> buat file hello.gblk  
> isi dengan di bawah ini    

```gblk
tulis("Hello world!")
```

> run dengan menjalankan command    
**[gblk binary exe] [nama file]**    

`./gblk.exe hello.gblk`

## Output (STDOUT):    
"Hello world"
